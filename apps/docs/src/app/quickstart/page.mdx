This guide will get you all set up and ready to use the GLAPI accounting dimensions API. Follow these steps to make your first API call. {{ className: 'lead' }}

## Make your first API call

Use the API key in the `X-API-Key` header to authenticate your requests:

<CodeGroup>

```bash {{ title: 'cURL' }}
curl -X GET https://api.glapi.io/api/v1/customers \
  -H "X-API-Key: glapi_test_sk_1234567890abcdef" \
  -H "Content-Type: application/json"
```

```javascript {{ title: 'JavaScript' }}
const response = await fetch('https://api.glapi.io/api/v1/customers', {
  headers: {
    'X-API-Key': 'glapi_test_sk_1234567890abcdef',
    'Content-Type': 'application/json'
  }
});

const customers = await response.json();
console.log(customers);
```

```python {{ title: 'Python' }}
import requests

headers = {
    'X-API-Key': 'glapi_test_sk_1234567890abcdef',
    'Content-Type': 'application/json'
}

response = requests.get('https://api.glapi.io/api/v1/customers', headers=headers)
customers = response.json()
print(customers)
```

</CodeGroup>

## Available endpoints

The GLAPI provides endpoints for managing accounting dimensions:

- **Customers** - `/api/v1/customers`
- **Vendors** - `/api/v1/vendors`
- **Employees** - `/api/v1/employees`
- **Departments** - `/api/v1/departments`
- **Locations** - `/api/v1/locations`
- **Classes** - `/api/v1/classes`
- **Subsidiaries** - `/api/v1/subsidiaries`

Each endpoint supports standard CRUD operations (GET, POST, PUT, DELETE).

## Create a customer

Let's create your first customer using the API:

<CodeGroup tag="POST" label="/api/v1/customers">

```bash {{ title: 'cURL' }}
curl -X POST https://api.glapi.io/api/v1/customers \
  -H "X-API-Key: glapi_test_sk_1234567890abcdef" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Acme Corporation",
    "email": "contact@acme.com",
    "phone": "555-0123",
    "address": "123 Main St, Anytown, USA"
  }'
```

```javascript {{ title: 'JavaScript' }}
const customer = {
  name: "Acme Corporation",
  email: "contact@acme.com",
  phone: "555-0123",
  address: "123 Main St, Anytown, USA"
};

const response = await fetch('https://api.glapi.io/api/v1/customers', {
  method: 'POST',
  headers: {
    'X-API-Key': 'glapi_test_sk_1234567890abcdef',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify(customer)
});

const newCustomer = await response.json();
console.log(newCustomer);
```

```python {{ title: 'Python' }}
import requests

customer = {
    "name": "Acme Corporation",
    "email": "contact@acme.com",
    "phone": "555-0123",
    "address": "123 Main St, Anytown, USA"
}

headers = {
    'X-API-Key': 'glapi_test_sk_1234567890abcdef',
    'Content-Type': 'application/json'
}

response = requests.post('https://api.glapi.io/api/v1/customers', 
                        json=customer, headers=headers)
new_customer = response.json()
print(new_customer)
```

</CodeGroup>
