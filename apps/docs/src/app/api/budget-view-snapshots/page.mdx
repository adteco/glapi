# Budget View Snapshots API

The Budget View Snapshots API allows you to retrieve a list of budget view snapshots for a specific project. Snapshots represent the state of a budget view at a particular point in time. {{ className: 'lead' }}

## Get Budget View Snapshots

<CodeGroup tag="GET" label="/api/external/budget-view-snapshots">

```bash
curl -X GET "https://api.kurrent.com/api/external/budget-view-snapshots?project_id=123" \
  -H "x-api-key: your-api-key" \
  -H "x-organization-id: your-organization-id" \
  -H "x-company-id: your-company-id"
```

</CodeGroup>

### Query Parameters

<Properties>
  <Property name="project_id" type="integer" required>
    The ID of the project for which to retrieve budget view snapshots.
  </Property>
</Properties>

### Required Headers

<Properties>
  <Property name="x-api-key" type="string" required>
    Your API key for authentication.
  </Property>
  <Property name="x-organization-id" type="string" required>
    The ID of your organization.
  </Property>
  <Property name="x-company-id" type="string" required>
    The ID of the company you're accessing.
  </Property>
</Properties>

### Response

<ResponseField name="data" type="array">
  An array of budget view snapshot objects.

  <Expandable title="Budget View Snapshot Object">
    <ResponseField name="id" type="integer">
      The unique identifier for the budget view snapshot.
    </ResponseField>
    <ResponseField name="name" type="string">
      The name of the snapshot.
    </ResponseField>
    <ResponseField name="description" type="string">
      A description of the snapshot.
    </ResponseField>
    <ResponseField name="created_at" type="string">
      The date and time when the snapshot was created, in ISO 8601 format.
    </ResponseField>
    <ResponseField name="budget_view_id" type="integer">
      The ID of the budget view that this snapshot belongs to.
    </ResponseField>
    <ResponseField name="project_id" type="integer">
      The ID of the project that this snapshot belongs to.
    </ResponseField>
    <ResponseField name="created_by" type="object">
      Information about the user who created the snapshot.
      <Expandable title="User Object">
        <ResponseField name="id" type="integer">
          The unique identifier for the user.
        </ResponseField>
        <ResponseField name="name" type="string">
          The name of the user.
        </ResponseField>
        <ResponseField name="email" type="string">
          The email address of the user.
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>
