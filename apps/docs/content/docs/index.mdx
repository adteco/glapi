# GLAPI Documentation

Welcome to the official documentation for **GLAPI** — an API-first General Ledger platform designed for modern SaaS companies with complex revenue recognition requirements.

## Overview

GLAPI provides a complete accounting API that handles:

- **Double-Entry Accounting**: Full journal entry support with automatic balancing and audit trails
- **Revenue Recognition (ASC 606)**: Automated revenue recognition with performance obligations, SSP allocation, and compliance reporting
- **Multi-Entity Support**: Manage subsidiaries, departments, locations, and classes
- **Subscription Management**: Track recurring revenue, billing schedules, and customer lifecycle
- **Real-time Reporting**: Generate balance sheets, income statements, and custom financial reports
