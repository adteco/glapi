# Locations API

Manage locations in your organization.

## Base Path

```
/api/locations
```

## Endpoints

### List Locations

List all locations in your organization.

```http
GET /api/locations
```

**Query Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `includeInactive` | boolean | No | Include inactive records in the results |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/locations
```

**Example Response:**

```json
[
  {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "organizationId": "org_123",
    "status": "active",
    "createdAt": "2025-01-15T10:30:00Z",
    "updatedAt": "2025-01-15T10:30:00Z"
  }
]
```

---

### Get Location

Retrieve a specific location by ID.

```http
GET /api/locations/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Location ID |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/locations/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Location not found

---

### Create Location

Create a new location.

```http
POST /api/locations
```

**Request Body:**

```json
{
  "status": "active"
}
```

**Example Request:**

```bash
curl -X POST \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "active"}' \
  https://api.glapi.io/api/locations
```

---

### Update Location

Update an existing location.

```http
PUT /api/locations/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Location ID |

**Request Body:**

Provide only the fields you want to update. All fields are optional.

```json
{
  "status": "inactive"
}
```

**Example Request:**

```bash
curl -X PUT \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "inactive"}' \
  https://api.glapi.io/api/locations/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Location not found

---

### Delete Location

Delete a location.

```http
DELETE /api/locations/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Location ID |

**Example Request:**

```bash
curl -X DELETE \
  -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/locations/123e4567-e89b-12d3-a456-426614174000
```

**Example Response:**

```json
{
  "success": true
}
```

**Error Responses:**

- `404 Not Found` - Location not found

## Code Examples

### TypeScript

```typescript
import { createTRPCClient, httpBatchLink } from '@trpc/client';
import type { AppRouter } from '@glapi/trpc';

const client = createTRPCClient<AppRouter>({
  links: [
    httpBatchLink({
      url: 'https://api.glapi.io/api/trpc',
      headers: () => ({
        Authorization: `Bearer ${process.env.GLAPI_TOKEN}`,
      }),
    }),
  ],
});

// List locations
const items = await client.locations.list.query();

// Get location
const item = await client.locations.get.query({
  id: '123e4567-e89b-12d3-a456-426614174000'
});

// Create location
const newItem = await client.locations.create.mutate({
  status: 'active',
});

// Update location
const updated = await client.locations.update.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000',
  data: {
    status: 'inactive',
  },
});

// Delete location
await client.locations.delete.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000'
});
```

### Python

```python
import requests
import os

API_BASE = 'https://api.glapi.io/api'
TOKEN = os.environ.get('GLAPI_TOKEN')
HEADERS = {
    'Authorization': f'Bearer {TOKEN}',
    'Content-Type': 'application/json',
}

# List locations
response = requests.get(f'{API_BASE}/locations', headers=HEADERS)
items = response.json()

# Get location
response = requests.get(
    f'{API_BASE}/locations/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
item = response.json()

# Create location
response = requests.post(
    f'{API_BASE}/locations',
    headers=HEADERS,
    json={'status': 'active'}
)
new_item = response.json()

# Update location
response = requests.put(
    f'{API_BASE}/locations/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS,
    json={'status': 'inactive'}
)
updated = response.json()

# Delete location
response = requests.delete(
    f'{API_BASE}/locations/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
```
