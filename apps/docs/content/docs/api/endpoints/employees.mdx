# Employees API

Manage employees in your organization.

## Base Path

```
/api/employees
```

## Endpoints

### List Employees

List all employees in your organization.

```http
GET /api/employees
```

**Query Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `includeInactive` | boolean | No | Include inactive records in the results |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/employees
```

**Example Response:**

```json
[
  {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "organizationId": "org_123",
    "status": "active",
    "createdAt": "2025-01-15T10:30:00Z",
    "updatedAt": "2025-01-15T10:30:00Z"
  }
]
```

---

### Get Employee

Retrieve a specific employee by ID.

```http
GET /api/employees/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Employee ID |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/employees/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Employee not found

---

### Create Employee

Create a new employee.

```http
POST /api/employees
```

**Request Body:**

```json
{
  "status": "active"
}
```

**Example Request:**

```bash
curl -X POST \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "active"}' \
  https://api.glapi.io/api/employees
```

---

### Update Employee

Update an existing employee.

```http
PUT /api/employees/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Employee ID |

**Request Body:**

Provide only the fields you want to update. All fields are optional.

```json
{
  "status": "inactive"
}
```

**Example Request:**

```bash
curl -X PUT \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "inactive"}' \
  https://api.glapi.io/api/employees/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Employee not found

---

### Delete Employee

Delete a employee.

```http
DELETE /api/employees/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Employee ID |

**Example Request:**

```bash
curl -X DELETE \
  -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/employees/123e4567-e89b-12d3-a456-426614174000
```

**Example Response:**

```json
{
  "success": true
}
```

**Error Responses:**

- `404 Not Found` - Employee not found

## Code Examples

### TypeScript

```typescript
import { createTRPCClient, httpBatchLink } from '@trpc/client';
import type { AppRouter } from '@glapi/trpc';

const client = createTRPCClient<AppRouter>({
  links: [
    httpBatchLink({
      url: 'https://api.glapi.io/api/trpc',
      headers: () => ({
        Authorization: `Bearer ${process.env.GLAPI_TOKEN}`,
      }),
    }),
  ],
});

// List employees
const items = await client.employees.list.query();

// Get employee
const item = await client.employees.get.query({
  id: '123e4567-e89b-12d3-a456-426614174000'
});

// Create employee
const newItem = await client.employees.create.mutate({
  status: 'active',
});

// Update employee
const updated = await client.employees.update.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000',
  data: {
    status: 'inactive',
  },
});

// Delete employee
await client.employees.delete.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000'
});
```

### Python

```python
import requests
import os

API_BASE = 'https://api.glapi.io/api'
TOKEN = os.environ.get('GLAPI_TOKEN')
HEADERS = {
    'Authorization': f'Bearer {TOKEN}',
    'Content-Type': 'application/json',
}

# List employees
response = requests.get(f'{API_BASE}/employees', headers=HEADERS)
items = response.json()

# Get employee
response = requests.get(
    f'{API_BASE}/employees/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
item = response.json()

# Create employee
response = requests.post(
    f'{API_BASE}/employees',
    headers=HEADERS,
    json={'status': 'active'}
)
new_item = response.json()

# Update employee
response = requests.put(
    f'{API_BASE}/employees/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS,
    json={'status': 'inactive'}
)
updated = response.json()

# Delete employee
response = requests.delete(
    f'{API_BASE}/employees/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
```
