# Price Lists API

Manage price-lists in your organization.

## Base Path

```
/api/price-lists
```

## Endpoints

### List Price Lists

List all price-lists in your organization.

```http
GET /api/price-lists
```

**Query Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `includeInactive` | boolean | No | Include inactive records in the results |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/price-lists
```

**Example Response:**

```json
[
  {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "organizationId": "org_123",
    "status": "active",
    "createdAt": "2025-01-15T10:30:00Z",
    "updatedAt": "2025-01-15T10:30:00Z"
  }
]
```

---

### Get Price list

Retrieve a specific price list by ID.

```http
GET /api/price-lists/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Price list ID |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/price-lists/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Price list not found

---

### Create Price list

Create a new price list.

```http
POST /api/price-lists
```

**Request Body:**

```json
{
  "status": "active"
}
```

**Example Request:**

```bash
curl -X POST \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "active"}' \
  https://api.glapi.io/api/price-lists
```

---

### Update Price list

Update an existing price list.

```http
PUT /api/price-lists/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Price list ID |

**Request Body:**

Provide only the fields you want to update. All fields are optional.

```json
{
  "status": "inactive"
}
```

**Example Request:**

```bash
curl -X PUT \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "inactive"}' \
  https://api.glapi.io/api/price-lists/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Price list not found

---

### Delete Price list

Delete a price list.

```http
DELETE /api/price-lists/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Price list ID |

**Example Request:**

```bash
curl -X DELETE \
  -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/price-lists/123e4567-e89b-12d3-a456-426614174000
```

**Example Response:**

```json
{
  "success": true
}
```

**Error Responses:**

- `404 Not Found` - Price list not found

## Code Examples

### TypeScript

```typescript
import { createTRPCClient, httpBatchLink } from '@trpc/client';
import type { AppRouter } from '@glapi/trpc';

const client = createTRPCClient<AppRouter>({
  links: [
    httpBatchLink({
      url: 'https://api.glapi.io/api/trpc',
      headers: () => ({
        Authorization: `Bearer ${process.env.GLAPI_TOKEN}`,
      }),
    }),
  ],
});

// List price-lists
const items = await client.pricelists.list.query();

// Get price list
const item = await client.pricelists.get.query({
  id: '123e4567-e89b-12d3-a456-426614174000'
});

// Create price list
const newItem = await client.pricelists.create.mutate({
  status: 'active',
});

// Update price list
const updated = await client.pricelists.update.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000',
  data: {
    status: 'inactive',
  },
});

// Delete price list
await client.pricelists.delete.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000'
});
```

### Python

```python
import requests
import os

API_BASE = 'https://api.glapi.io/api'
TOKEN = os.environ.get('GLAPI_TOKEN')
HEADERS = {
    'Authorization': f'Bearer {TOKEN}',
    'Content-Type': 'application/json',
}

# List price-lists
response = requests.get(f'{API_BASE}/price-lists', headers=HEADERS)
items = response.json()

# Get price list
response = requests.get(
    f'{API_BASE}/price-lists/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
item = response.json()

# Create price list
response = requests.post(
    f'{API_BASE}/price-lists',
    headers=HEADERS,
    json={'status': 'active'}
)
new_item = response.json()

# Update price list
response = requests.put(
    f'{API_BASE}/price-lists/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS,
    json={'status': 'inactive'}
)
updated = response.json()

# Delete price list
response = requests.delete(
    f'{API_BASE}/price-lists/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
```
