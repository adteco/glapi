# Warehouses API

Manage warehouses in your organization.

## Base Path

```
/api/warehouses
```

## Endpoints

### List Warehouses

List all warehouses in your organization.

```http
GET /api/warehouses
```

**Query Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `includeInactive` | boolean | No | Include inactive records in the results |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/warehouses
```

**Example Response:**

```json
[
  {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "organizationId": "org_123",
    "status": "active",
    "createdAt": "2025-01-15T10:30:00Z",
    "updatedAt": "2025-01-15T10:30:00Z"
  }
]
```

---

### Get Warehouse

Retrieve a specific warehouse by ID.

```http
GET /api/warehouses/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Warehouse ID |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/warehouses/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Warehouse not found

---

### Create Warehouse

Create a new warehouse.

```http
POST /api/warehouses
```

**Request Body:**

```json
{
  "status": "active"
}
```

**Example Request:**

```bash
curl -X POST \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "active"}' \
  https://api.glapi.io/api/warehouses
```

---

### Update Warehouse

Update an existing warehouse.

```http
PUT /api/warehouses/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Warehouse ID |

**Request Body:**

Provide only the fields you want to update. All fields are optional.

```json
{
  "status": "inactive"
}
```

**Example Request:**

```bash
curl -X PUT \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "inactive"}' \
  https://api.glapi.io/api/warehouses/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Warehouse not found

---

### Delete Warehouse

Delete a warehouse.

```http
DELETE /api/warehouses/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Warehouse ID |

**Example Request:**

```bash
curl -X DELETE \
  -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/warehouses/123e4567-e89b-12d3-a456-426614174000
```

**Example Response:**

```json
{
  "success": true
}
```

**Error Responses:**

- `404 Not Found` - Warehouse not found

## Code Examples

### TypeScript

```typescript
import { createTRPCClient, httpBatchLink } from '@trpc/client';
import type { AppRouter } from '@glapi/trpc';

const client = createTRPCClient<AppRouter>({
  links: [
    httpBatchLink({
      url: 'https://api.glapi.io/api/trpc',
      headers: () => ({
        Authorization: `Bearer ${process.env.GLAPI_TOKEN}`,
      }),
    }),
  ],
});

// List warehouses
const items = await client.warehouses.list.query();

// Get warehouse
const item = await client.warehouses.get.query({
  id: '123e4567-e89b-12d3-a456-426614174000'
});

// Create warehouse
const newItem = await client.warehouses.create.mutate({
  status: 'active',
});

// Update warehouse
const updated = await client.warehouses.update.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000',
  data: {
    status: 'inactive',
  },
});

// Delete warehouse
await client.warehouses.delete.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000'
});
```

### Python

```python
import requests
import os

API_BASE = 'https://api.glapi.io/api'
TOKEN = os.environ.get('GLAPI_TOKEN')
HEADERS = {
    'Authorization': f'Bearer {TOKEN}',
    'Content-Type': 'application/json',
}

# List warehouses
response = requests.get(f'{API_BASE}/warehouses', headers=HEADERS)
items = response.json()

# Get warehouse
response = requests.get(
    f'{API_BASE}/warehouses/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
item = response.json()

# Create warehouse
response = requests.post(
    f'{API_BASE}/warehouses',
    headers=HEADERS,
    json={'status': 'active'}
)
new_item = response.json()

# Update warehouse
response = requests.put(
    f'{API_BASE}/warehouses/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS,
    json={'status': 'inactive'}
)
updated = response.json()

# Delete warehouse
response = requests.delete(
    f'{API_BASE}/warehouses/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
```
