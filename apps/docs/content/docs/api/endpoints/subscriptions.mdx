# Subscriptions API

Manage subscriptions in your organization.

## Base Path

```
/api/subscriptions
```

## Endpoints

### List Subscriptions

List all subscriptions in your organization.

```http
GET /api/subscriptions
```

**Query Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `includeInactive` | boolean | No | Include inactive records in the results |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/subscriptions
```

**Example Response:**

```json
[
  {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "organizationId": "org_123",
    "status": "active",
    "createdAt": "2025-01-15T10:30:00Z",
    "updatedAt": "2025-01-15T10:30:00Z"
  }
]
```

---

### Get Subscription

Retrieve a specific subscription by ID.

```http
GET /api/subscriptions/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Subscription ID |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/subscriptions/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Subscription not found

---

### Create Subscription

Create a new subscription.

```http
POST /api/subscriptions
```

**Request Body:**

```json
{
  "status": "active"
}
```

**Example Request:**

```bash
curl -X POST \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "active"}' \
  https://api.glapi.io/api/subscriptions
```

---

### Update Subscription

Update an existing subscription.

```http
PUT /api/subscriptions/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Subscription ID |

**Request Body:**

Provide only the fields you want to update. All fields are optional.

```json
{
  "status": "inactive"
}
```

**Example Request:**

```bash
curl -X PUT \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "inactive"}' \
  https://api.glapi.io/api/subscriptions/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Subscription not found

---

### Delete Subscription

Delete a subscription.

```http
DELETE /api/subscriptions/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Subscription ID |

**Example Request:**

```bash
curl -X DELETE \
  -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/subscriptions/123e4567-e89b-12d3-a456-426614174000
```

**Example Response:**

```json
{
  "success": true
}
```

**Error Responses:**

- `404 Not Found` - Subscription not found

## Code Examples

### TypeScript

```typescript
import { createTRPCClient, httpBatchLink } from '@trpc/client';
import type { AppRouter } from '@glapi/trpc';

const client = createTRPCClient<AppRouter>({
  links: [
    httpBatchLink({
      url: 'https://api.glapi.io/api/trpc',
      headers: () => ({
        Authorization: `Bearer ${process.env.GLAPI_TOKEN}`,
      }),
    }),
  ],
});

// List subscriptions
const items = await client.subscriptions.list.query();

// Get subscription
const item = await client.subscriptions.get.query({
  id: '123e4567-e89b-12d3-a456-426614174000'
});

// Create subscription
const newItem = await client.subscriptions.create.mutate({
  status: 'active',
});

// Update subscription
const updated = await client.subscriptions.update.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000',
  data: {
    status: 'inactive',
  },
});

// Delete subscription
await client.subscriptions.delete.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000'
});
```

### Python

```python
import requests
import os

API_BASE = 'https://api.glapi.io/api'
TOKEN = os.environ.get('GLAPI_TOKEN')
HEADERS = {
    'Authorization': f'Bearer {TOKEN}',
    'Content-Type': 'application/json',
}

# List subscriptions
response = requests.get(f'{API_BASE}/subscriptions', headers=HEADERS)
items = response.json()

# Get subscription
response = requests.get(
    f'{API_BASE}/subscriptions/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
item = response.json()

# Create subscription
response = requests.post(
    f'{API_BASE}/subscriptions',
    headers=HEADERS,
    json={'status': 'active'}
)
new_item = response.json()

# Update subscription
response = requests.put(
    f'{API_BASE}/subscriptions/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS,
    json={'status': 'inactive'}
)
updated = response.json()

# Delete subscription
response = requests.delete(
    f'{API_BASE}/subscriptions/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
```
