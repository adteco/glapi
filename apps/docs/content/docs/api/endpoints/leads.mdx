# Leads API

Manage leads in your organization.

## Base Path

```
/api/leads
```

## Endpoints

### List Leads

List all leads in your organization.

```http
GET /api/leads
```

**Query Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `includeInactive` | boolean | No | Include inactive records in the results |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/leads
```

**Example Response:**

```json
[
  {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "organizationId": "org_123",
    "status": "active",
    "createdAt": "2025-01-15T10:30:00Z",
    "updatedAt": "2025-01-15T10:30:00Z"
  }
]
```

---

### Get Lead

Retrieve a specific lead by ID.

```http
GET /api/leads/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Lead ID |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/leads/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Lead not found

---

### Create Lead

Create a new lead.

```http
POST /api/leads
```

**Request Body:**

```json
{
  "status": "active"
}
```

**Example Request:**

```bash
curl -X POST \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "active"}' \
  https://api.glapi.io/api/leads
```

---

### Update Lead

Update an existing lead.

```http
PUT /api/leads/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Lead ID |

**Request Body:**

Provide only the fields you want to update. All fields are optional.

```json
{
  "status": "inactive"
}
```

**Example Request:**

```bash
curl -X PUT \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "inactive"}' \
  https://api.glapi.io/api/leads/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Lead not found

---

### Delete Lead

Delete a lead.

```http
DELETE /api/leads/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Lead ID |

**Example Request:**

```bash
curl -X DELETE \
  -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/leads/123e4567-e89b-12d3-a456-426614174000
```

**Example Response:**

```json
{
  "success": true
}
```

**Error Responses:**

- `404 Not Found` - Lead not found

## Code Examples

### TypeScript

```typescript
import { createTRPCClient, httpBatchLink } from '@trpc/client';
import type { AppRouter } from '@glapi/trpc';

const client = createTRPCClient<AppRouter>({
  links: [
    httpBatchLink({
      url: 'https://api.glapi.io/api/trpc',
      headers: () => ({
        Authorization: `Bearer ${process.env.GLAPI_TOKEN}`,
      }),
    }),
  ],
});

// List leads
const items = await client.leads.list.query();

// Get lead
const item = await client.leads.get.query({
  id: '123e4567-e89b-12d3-a456-426614174000'
});

// Create lead
const newItem = await client.leads.create.mutate({
  status: 'active',
});

// Update lead
const updated = await client.leads.update.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000',
  data: {
    status: 'inactive',
  },
});

// Delete lead
await client.leads.delete.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000'
});
```

### Python

```python
import requests
import os

API_BASE = 'https://api.glapi.io/api'
TOKEN = os.environ.get('GLAPI_TOKEN')
HEADERS = {
    'Authorization': f'Bearer {TOKEN}',
    'Content-Type': 'application/json',
}

# List leads
response = requests.get(f'{API_BASE}/leads', headers=HEADERS)
items = response.json()

# Get lead
response = requests.get(
    f'{API_BASE}/leads/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
item = response.json()

# Create lead
response = requests.post(
    f'{API_BASE}/leads',
    headers=HEADERS,
    json={'status': 'active'}
)
new_item = response.json()

# Update lead
response = requests.put(
    f'{API_BASE}/leads/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS,
    json={'status': 'inactive'}
)
updated = response.json()

# Delete lead
response = requests.delete(
    f'{API_BASE}/leads/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
```
