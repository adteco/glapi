# Prospects API

Manage prospects in your organization.

## Base Path

```
/api/prospects
```

## Endpoints

### List Prospects

List all prospects in your organization.

```http
GET /api/prospects
```

**Query Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `includeInactive` | boolean | No | Include inactive records in the results |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/prospects
```

**Example Response:**

```json
[
  {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "organizationId": "org_123",
    "status": "active",
    "createdAt": "2025-01-15T10:30:00Z",
    "updatedAt": "2025-01-15T10:30:00Z"
  }
]
```

---

### Get Prospect

Retrieve a specific prospect by ID.

```http
GET /api/prospects/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Prospect ID |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/prospects/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Prospect not found

---

### Create Prospect

Create a new prospect.

```http
POST /api/prospects
```

**Request Body:**

```json
{
  "status": "active"
}
```

**Example Request:**

```bash
curl -X POST \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "active"}' \
  https://api.glapi.io/api/prospects
```

---

### Update Prospect

Update an existing prospect.

```http
PUT /api/prospects/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Prospect ID |

**Request Body:**

Provide only the fields you want to update. All fields are optional.

```json
{
  "status": "inactive"
}
```

**Example Request:**

```bash
curl -X PUT \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "inactive"}' \
  https://api.glapi.io/api/prospects/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Prospect not found

---

### Delete Prospect

Delete a prospect.

```http
DELETE /api/prospects/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Prospect ID |

**Example Request:**

```bash
curl -X DELETE \
  -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/prospects/123e4567-e89b-12d3-a456-426614174000
```

**Example Response:**

```json
{
  "success": true
}
```

**Error Responses:**

- `404 Not Found` - Prospect not found

## Code Examples

### TypeScript

```typescript
import { createTRPCClient, httpBatchLink } from '@trpc/client';
import type { AppRouter } from '@glapi/trpc';

const client = createTRPCClient<AppRouter>({
  links: [
    httpBatchLink({
      url: 'https://api.glapi.io/api/trpc',
      headers: () => ({
        Authorization: `Bearer ${process.env.GLAPI_TOKEN}`,
      }),
    }),
  ],
});

// List prospects
const items = await client.prospects.list.query();

// Get prospect
const item = await client.prospects.get.query({
  id: '123e4567-e89b-12d3-a456-426614174000'
});

// Create prospect
const newItem = await client.prospects.create.mutate({
  status: 'active',
});

// Update prospect
const updated = await client.prospects.update.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000',
  data: {
    status: 'inactive',
  },
});

// Delete prospect
await client.prospects.delete.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000'
});
```

### Python

```python
import requests
import os

API_BASE = 'https://api.glapi.io/api'
TOKEN = os.environ.get('GLAPI_TOKEN')
HEADERS = {
    'Authorization': f'Bearer {TOKEN}',
    'Content-Type': 'application/json',
}

# List prospects
response = requests.get(f'{API_BASE}/prospects', headers=HEADERS)
items = response.json()

# Get prospect
response = requests.get(
    f'{API_BASE}/prospects/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
item = response.json()

# Create prospect
response = requests.post(
    f'{API_BASE}/prospects',
    headers=HEADERS,
    json={'status': 'active'}
)
new_item = response.json()

# Update prospect
response = requests.put(
    f'{API_BASE}/prospects/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS,
    json={'status': 'inactive'}
)
updated = response.json()

# Delete prospect
response = requests.delete(
    f'{API_BASE}/prospects/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
```
