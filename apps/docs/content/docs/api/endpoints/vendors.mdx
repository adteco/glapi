# Vendors API

Manage vendors in your organization.

## Base Path

```
/api/vendors
```

## Endpoints

### List Vendors

List all vendors in your organization.

```http
GET /api/vendors
```

**Query Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `includeInactive` | boolean | No | Include inactive records in the results |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/vendors
```

**Example Response:**

```json
[
  {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "organizationId": "org_123",
    "status": "active",
    "createdAt": "2025-01-15T10:30:00Z",
    "updatedAt": "2025-01-15T10:30:00Z"
  }
]
```

---

### Get Vendor

Retrieve a specific vendor by ID.

```http
GET /api/vendors/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Vendor ID |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/vendors/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Vendor not found

---

### Create Vendor

Create a new vendor.

```http
POST /api/vendors
```

**Request Body:**

```json
{
  "status": "active"
}
```

**Example Request:**

```bash
curl -X POST \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "active"}' \
  https://api.glapi.io/api/vendors
```

---

### Update Vendor

Update an existing vendor.

```http
PUT /api/vendors/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Vendor ID |

**Request Body:**

Provide only the fields you want to update. All fields are optional.

```json
{
  "status": "inactive"
}
```

**Example Request:**

```bash
curl -X PUT \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "inactive"}' \
  https://api.glapi.io/api/vendors/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Vendor not found

---

### Delete Vendor

Delete a vendor.

```http
DELETE /api/vendors/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Vendor ID |

**Example Request:**

```bash
curl -X DELETE \
  -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/vendors/123e4567-e89b-12d3-a456-426614174000
```

**Example Response:**

```json
{
  "success": true
}
```

**Error Responses:**

- `404 Not Found` - Vendor not found

## Code Examples

### TypeScript

```typescript
import { createTRPCClient, httpBatchLink } from '@trpc/client';
import type { AppRouter } from '@glapi/trpc';

const client = createTRPCClient<AppRouter>({
  links: [
    httpBatchLink({
      url: 'https://api.glapi.io/api/trpc',
      headers: () => ({
        Authorization: `Bearer ${process.env.GLAPI_TOKEN}`,
      }),
    }),
  ],
});

// List vendors
const items = await client.vendors.list.query();

// Get vendor
const item = await client.vendors.get.query({
  id: '123e4567-e89b-12d3-a456-426614174000'
});

// Create vendor
const newItem = await client.vendors.create.mutate({
  status: 'active',
});

// Update vendor
const updated = await client.vendors.update.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000',
  data: {
    status: 'inactive',
  },
});

// Delete vendor
await client.vendors.delete.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000'
});
```

### Python

```python
import requests
import os

API_BASE = 'https://api.glapi.io/api'
TOKEN = os.environ.get('GLAPI_TOKEN')
HEADERS = {
    'Authorization': f'Bearer {TOKEN}',
    'Content-Type': 'application/json',
}

# List vendors
response = requests.get(f'{API_BASE}/vendors', headers=HEADERS)
items = response.json()

# Get vendor
response = requests.get(
    f'{API_BASE}/vendors/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
item = response.json()

# Create vendor
response = requests.post(
    f'{API_BASE}/vendors',
    headers=HEADERS,
    json={'status': 'active'}
)
new_item = response.json()

# Update vendor
response = requests.put(
    f'{API_BASE}/vendors/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS,
    json={'status': 'inactive'}
)
updated = response.json()

# Delete vendor
response = requests.delete(
    f'{API_BASE}/vendors/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
```
