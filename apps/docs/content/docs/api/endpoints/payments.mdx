# Payments API

Manage payments in your organization.

## Base Path

```
/api/payments
```

## Endpoints

### List Payments

List all payments in your organization.

```http
GET /api/payments
```

**Query Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `includeInactive` | boolean | No | Include inactive records in the results |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/payments
```

**Example Response:**

```json
[
  {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "organizationId": "org_123",
    "status": "active",
    "createdAt": "2025-01-15T10:30:00Z",
    "updatedAt": "2025-01-15T10:30:00Z"
  }
]
```

---

### Get Payment

Retrieve a specific payment by ID.

```http
GET /api/payments/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Payment ID |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/payments/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Payment not found

---

### Create Payment

Create a new payment.

```http
POST /api/payments
```

**Request Body:**

```json
{
  "status": "active"
}
```

**Example Request:**

```bash
curl -X POST \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "active"}' \
  https://api.glapi.io/api/payments
```

---

### Update Payment

Update an existing payment.

```http
PUT /api/payments/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Payment ID |

**Request Body:**

Provide only the fields you want to update. All fields are optional.

```json
{
  "status": "inactive"
}
```

**Example Request:**

```bash
curl -X PUT \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "inactive"}' \
  https://api.glapi.io/api/payments/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Payment not found

---

### Delete Payment

Delete a payment.

```http
DELETE /api/payments/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Payment ID |

**Example Request:**

```bash
curl -X DELETE \
  -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/payments/123e4567-e89b-12d3-a456-426614174000
```

**Example Response:**

```json
{
  "success": true
}
```

**Error Responses:**

- `404 Not Found` - Payment not found

## Code Examples

### TypeScript

```typescript
import { createTRPCClient, httpBatchLink } from '@trpc/client';
import type { AppRouter } from '@glapi/trpc';

const client = createTRPCClient<AppRouter>({
  links: [
    httpBatchLink({
      url: 'https://api.glapi.io/api/trpc',
      headers: () => ({
        Authorization: `Bearer ${process.env.GLAPI_TOKEN}`,
      }),
    }),
  ],
});

// List payments
const items = await client.payments.list.query();

// Get payment
const item = await client.payments.get.query({
  id: '123e4567-e89b-12d3-a456-426614174000'
});

// Create payment
const newItem = await client.payments.create.mutate({
  status: 'active',
});

// Update payment
const updated = await client.payments.update.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000',
  data: {
    status: 'inactive',
  },
});

// Delete payment
await client.payments.delete.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000'
});
```

### Python

```python
import requests
import os

API_BASE = 'https://api.glapi.io/api'
TOKEN = os.environ.get('GLAPI_TOKEN')
HEADERS = {
    'Authorization': f'Bearer {TOKEN}',
    'Content-Type': 'application/json',
}

# List payments
response = requests.get(f'{API_BASE}/payments', headers=HEADERS)
items = response.json()

# Get payment
response = requests.get(
    f'{API_BASE}/payments/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
item = response.json()

# Create payment
response = requests.post(
    f'{API_BASE}/payments',
    headers=HEADERS,
    json={'status': 'active'}
)
new_item = response.json()

# Update payment
response = requests.put(
    f'{API_BASE}/payments/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS,
    json={'status': 'inactive'}
)
updated = response.json()

# Delete payment
response = requests.delete(
    f'{API_BASE}/payments/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
```
