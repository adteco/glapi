# Departments API

Manage departments in your organization.

## Base Path

```
/api/departments
```

## Endpoints

### List Departments

List all departments in your organization.

```http
GET /api/departments
```

**Query Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `includeInactive` | boolean | No | Include inactive records in the results |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/departments
```

**Example Response:**

```json
[
  {
    "id": "123e4567-e89b-12d3-a456-426614174000",
    "organizationId": "org_123",
    "status": "active",
    "createdAt": "2025-01-15T10:30:00Z",
    "updatedAt": "2025-01-15T10:30:00Z"
  }
]
```

---

### Get Department

Retrieve a specific department by ID.

```http
GET /api/departments/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Department ID |

**Example Request:**

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/departments/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Department not found

---

### Create Department

Create a new department.

```http
POST /api/departments
```

**Request Body:**

```json
{
  "status": "active"
}
```

**Example Request:**

```bash
curl -X POST \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "active"}' \
  https://api.glapi.io/api/departments
```

---

### Update Department

Update an existing department.

```http
PUT /api/departments/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Department ID |

**Request Body:**

Provide only the fields you want to update. All fields are optional.

```json
{
  "status": "inactive"
}
```

**Example Request:**

```bash
curl -X PUT \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "inactive"}' \
  https://api.glapi.io/api/departments/123e4567-e89b-12d3-a456-426614174000
```

**Error Responses:**

- `404 Not Found` - Department not found

---

### Delete Department

Delete a department.

```http
DELETE /api/departments/{id}
```

**Path Parameters:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | UUID | Yes | Department ID |

**Example Request:**

```bash
curl -X DELETE \
  -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.glapi.io/api/departments/123e4567-e89b-12d3-a456-426614174000
```

**Example Response:**

```json
{
  "success": true
}
```

**Error Responses:**

- `404 Not Found` - Department not found

## Code Examples

### TypeScript

```typescript
import { createTRPCClient, httpBatchLink } from '@trpc/client';
import type { AppRouter } from '@glapi/trpc';

const client = createTRPCClient<AppRouter>({
  links: [
    httpBatchLink({
      url: 'https://api.glapi.io/api/trpc',
      headers: () => ({
        Authorization: `Bearer ${process.env.GLAPI_TOKEN}`,
      }),
    }),
  ],
});

// List departments
const items = await client.departments.list.query();

// Get department
const item = await client.departments.get.query({
  id: '123e4567-e89b-12d3-a456-426614174000'
});

// Create department
const newItem = await client.departments.create.mutate({
  status: 'active',
});

// Update department
const updated = await client.departments.update.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000',
  data: {
    status: 'inactive',
  },
});

// Delete department
await client.departments.delete.mutate({
  id: '123e4567-e89b-12d3-a456-426614174000'
});
```

### Python

```python
import requests
import os

API_BASE = 'https://api.glapi.io/api'
TOKEN = os.environ.get('GLAPI_TOKEN')
HEADERS = {
    'Authorization': f'Bearer {TOKEN}',
    'Content-Type': 'application/json',
}

# List departments
response = requests.get(f'{API_BASE}/departments', headers=HEADERS)
items = response.json()

# Get department
response = requests.get(
    f'{API_BASE}/departments/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
item = response.json()

# Create department
response = requests.post(
    f'{API_BASE}/departments',
    headers=HEADERS,
    json={'status': 'active'}
)
new_item = response.json()

# Update department
response = requests.put(
    f'{API_BASE}/departments/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS,
    json={'status': 'inactive'}
)
updated = response.json()

# Delete department
response = requests.delete(
    f'{API_BASE}/departments/123e4567-e89b-12d3-a456-426614174000',
    headers=HEADERS
)
```
