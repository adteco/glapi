# Financial Statements API

Generate and export financial statements including Balance Sheet, Income Statement, Cash Flow Statement, and Trial Balance.

## Base Path

```
/api/financial-statements
```

## Endpoints

### Balance Sheet

Generate a Balance Sheet for the specified period showing assets, liabilities, and equity.

```http
POST /api/financial-statements/balanceSheet
```

**Request Body:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `periodId` | UUID | Yes | Accounting period ID |
| `comparePeriodId` | UUID | No | Period ID for comparative statement |
| `subsidiaryId` | UUID | No | Filter by subsidiary |
| `departmentIds` | UUID[] | No | Filter by departments |
| `classIds` | UUID[] | No | Filter by classes |
| `locationIds` | UUID[] | No | Filter by locations |
| `includeInactive` | boolean | No | Include inactive accounts (default: false) |
| `showAccountHierarchy` | boolean | No | Show account hierarchy (default: true) |
| `showZeroBalances` | boolean | No | Show accounts with zero balances (default: false) |

**Example Request:**

```bash
curl -X POST \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "periodId": "123e4567-e89b-12d3-a456-426614174000",
    "showZeroBalances": false
  }' \
  https://api.glapi.io/api/financial-statements/balanceSheet
```

### Export Report

Export a generated financial statement to PDF, Excel, CSV, or JSON format.

```http
POST /api/financial-statements/export
```

**Request Body:**

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `reportType` | string | Yes | Report type: `INCOME_STATEMENT`, `BALANCE_SHEET`, or `CASH_FLOW_STATEMENT` |
| `reportData` | object | Yes | The generated report data (from one of the above endpoints) |
| `format` | string | Yes | Export format: `pdf`, `xlsx`, `csv`, or `json` |
| `includeComparison` | boolean | No | Include comparison period (default: false) |
| `includeNotes` | boolean | No | Include footnotes (default: false) |
| `includeLogo` | boolean | No | Include company logo (default: true) |
| `landscape` | boolean | No | Use landscape orientation (default: false) |

**Example Request:**

```bash
curl -X POST \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "reportType": "INCOME_STATEMENT",
    "reportData": { ... },
    "format": "pdf",
    "landscape": true
  }' \
  https://api.glapi.io/api/financial-statements/export
```

**Example Response:**

```json
{
  "filename": "income_statement_2025-01-31.pdf",
  "contentType": "application/pdf",
  "content": "JVBERi0xLjQKJdPr6eEKMSAwI... (base64 encoded)"
}
```

## Code Examples

### TypeScript

```typescript
import { createTRPCClient, httpBatchLink } from '@trpc/client';
import type { AppRouter } from '@glapi/trpc';

const client = createTRPCClient<AppRouter>({
  links: [
    httpBatchLink({
      url: 'https://api.glapi.io/api/trpc',
      headers: () => ({
        Authorization: `Bearer ${process.env.GLAPI_TOKEN}`,
      }),
    }),
  ],
});

// Generate Balance Sheet
const balanceSheet = await client.financialStatements.balanceSheet.query({
  periodId: '123e4567-e89b-12d3-a456-426614174000',
  showZeroBalances: false,
});

// Generate Income Statement with comparison
const incomeStatement = await client.financialStatements.incomeStatement.query({
  periodId: '123e4567-e89b-12d3-a456-426614174000',
  comparePeriodId: '987e6543-e89b-12d3-a456-426614174000',
  includeYTD: true,
});

// Generate Cash Flow Statement
const cashFlow = await client.financialStatements.cashFlowStatement.query({
  periodId: '123e4567-e89b-12d3-a456-426614174000',
});

// Export to PDF
const pdfExport = await client.financialStatements.export.mutate({
  reportType: 'INCOME_STATEMENT',
  reportData: incomeStatement,
  format: 'pdf',
  landscape: true,
});

// Decode and save the PDF
const buffer = Buffer.from(pdfExport.content, 'base64');
fs.writeFileSync(pdfExport.filename, buffer);
```

### Python

```python
import requests
import base64
import os

API_BASE = 'https://api.glapi.io/api/financial-statements'
TOKEN = os.environ.get('GLAPI_TOKEN')
HEADERS = {
    'Authorization': f'Bearer {TOKEN}',
    'Content-Type': 'application/json',
}

# Generate Balance Sheet
response = requests.post(
    f'{API_BASE}/balanceSheet',
    headers=HEADERS,
    json={
        'periodId': '123e4567-e89b-12d3-a456-426614174000',
        'showZeroBalances': False
    }
)
balance_sheet = response.json()

# Generate Income Statement
response = requests.post(
    f'{API_BASE}/incomeStatement',
    headers=HEADERS,
    json={
        'periodId': '123e4567-e89b-12d3-a456-426614174000',
        'includeYTD': True
    }
)
income_statement = response.json()

# Export to Excel
response = requests.post(
    f'{API_BASE}/export',
    headers=HEADERS,
    json={
        'reportType': 'INCOME_STATEMENT',
        'reportData': income_statement,
        'format': 'xlsx'
    }
)
export_result = response.json()

# Save the Excel file
content = base64.b64decode(export_result['content'])
with open(export_result['filename'], 'wb') as f:
    f.write(content)
```
